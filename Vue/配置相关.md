# VSCode

- 解决@符的鼠标点击路径跳转问题

1. 在项目根目录下创建jsconfig.json文件
2. 重启VSCode

```json
{
  "compilerOptions": {
      "target": "ES6",
      "module": "commonjs",
      "allowSyntheticDefaultImports": true,
      "baseUrl": "./",
      "paths": {
        "@/*": ["src/*"]
      }
  },
  "exclude": [
      "node_modules"
  ]
}
```
